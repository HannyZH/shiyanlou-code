### 函数
默认参数值只能被赋值一次，若默认值是可变对象，例如列表，字典等，函数在后续调用中会累积前面传给它的参数。
```python
>>> def f(a, data=[]):
...    data.append(a)
...    return data
...
>>> print(f(1))
[1]
>>> print(f(2))
[1, 2]
>>> print(f(3))
[1, 2, 3]
```
要避免这个问题
```
>>> def f(a, data=None):
...     if data is None:
...         data = []
...     data.append(a)
...     return data
...
>>> print(f(1))
[1]
>>> print(f(2))
[2]
